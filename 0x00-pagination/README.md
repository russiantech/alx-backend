# Pagination Helper Function

This project contains a simple helper function for pagination in Python.

## Functionality

The `index_range` function calculates the start and end indexes for pagination based on the page number and page size.

## Usage

To use the `index_range` function, import it into your Python script and call it with the appropriate arguments:

```python
from simple_helper_function import index_range

start_index, end_index = index_range(page=1, page_size=10)
print(f"Start index: {start_index}, End index: {end_index}")
```

## Examples

Here are some examples of using the function:

- `index_range(1, 10)` returns `(0, 10)`
- `index_range(2, 15)` returns `(15, 30)`
